💓 CPR — Universal Steps (Adult / Child / Infant)
1
Unresponsive? Not breathing or only gasping → Shout for help, send someone for AED/defibrillator
2
Check pulse within 10 seconds. If present → Give 1 breath every 5–6 sec; check pulse every 2 min
3
IF NO PULSE → Position supine on hard flat surface. Begin 30:2 compressions, push hard & fast ≥100/min, allow full chest recoil — minimise interruptions
4
Open airway: head‑tilt / chin‑lift, ventilate ×2 (avoid excessive ventilations)
5
Attach AED (adult + child >1 yr). Shockable?
YES → Shock ×1, resume CPR 2 min NO → CPR 2 min + ALS interventions

📡 Cardiac Arrest Rhythms
Coarse VF — Chaotic irregular activity → SHOCK
Fine VF — Low‑amplitude irregular → SHOCK
Ventricular Tachycardia (VT) — Rapid wide complexes → SHOCK if pulseless
Asystole — Flat line, no activity → CPR only
PEA — Organised ECG but NO pulse → CPR only
📊 Common ECG Rhythms
✅ Normal Sinus RhythmRegular PQRST • PR 0.12–0.20s • QRS <0.12s • Rate 60–100
〰️ Atrial FibrillationIrregular rate • fibrillatory waves • no distinct P waves
⚡ SVTRapid narrow QRS • rate 150–250 • may have retrograde P
1° AV BlockPR interval >0.20 sec • all P waves conduct • benign
2° Mobitz I (Wenckebach)PR progressively lengthens → dropped QRS
⚠️ 2° Mobitz IIFixed PR, sudden dropped QRS — more dangerous
